Для сборки и запуска программы нужны Maven 3 и java8 (Прописаны в PATH)
1. С помощью консоли перейти в директорию проекта и выполнить команду: mvn clean package
2. В target директории проекта нужно запустить jar с помощью команды java -jar 
3. Приложение переводит английские слова в русские при помощи api
4. Для выхода из приложения написать "/" или ctrl+c
